Suppression de compte

Vous pouvez supprimer votre compte Elyra.

La suppression du compte est disponible directement dans l'app. Cette page explique les étapes, ce qui est supprimé et le contact à utiliser si vous rencontrez une difficulté.

Demander de l'aide Lire la confidentialité
Chemin dans l'app
  • Ouvrez Elyra.
  • Allez dans Profil ou Réglages.
  • Ouvrez Compte.
  • Choisissez Supprimer le compte et confirmez.
Étapes Données supprimées Conservation Aide

1. Comment supprimer votre compte depuis l'app

  1. Ouvrez l'application Elyra sur votre iPhone.
  2. Accédez à l'écran Profil ou Réglages.
  3. Ouvrez la section Compte.
  4. Sélectionnez Supprimer le compte.
  5. Lisez les informations affichées, puis confirmez la suppression.

Le libellé exact peut varier légèrement selon la version de l'app, mais la suppression restera accessible depuis la zone liée au compte.

2. Ce qui est supprimé

La suppression vise à effacer ou anonymiser les données liées à votre compte Elyra, notamment les informations de compte, les données nutrition, les séances, les modèles de séance, les données de communauté, les suppléments, les données de progression, l'hydratation, les préférences et les éléments de profil comme une photo si elle a été ajoutée.

3. Conservation minimale possible

Certaines informations peuvent être conservées pendant une durée limitée si cela est nécessaire pour respecter une obligation légale, assurer la sécurité du service, prévenir un abus, résoudre un litige ou conserver une trace technique minimale de la demande.

4. Si vous ne pouvez pas accéder à l'app

Si vous ne pouvez pas vous connecter ou si la suppression ne fonctionne pas, écrivez à support.elyra@gmail.com avec l'adresse associée au compte et une courte description du problème. Nous vous répondrons généralement sous 2 à 3 jours ouvrés.

5. Liens utiles

Pour comprendre plus largement les données traitées par Elyra, consultez la politique de confidentialité et les conditions d'utilisation.
